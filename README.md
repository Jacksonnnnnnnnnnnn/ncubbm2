# NCU 記錄系統
1st version(RecycleList done)

為 [NCU 研究記錄系統](https://github.com/Jacksonnnnnnnnnnnn/recordSystem) 開發的出版，僅有基本的 CRUD 功能，用於記錄研究用 sensors 之資訊，後來應用於 醫療資訊交流計劃 (雙和醫院神經外科與越南胡志明市大水鑊 (Cho Ray) 醫院)，因此界面更改為越南語言。
